# aicert-labs

Practical, executable examples that go with the study guides on [aicert.study](https://aicert.study) — a study platform for the four Anthropic Claude certifications (CCA-F, CCDV-F, CCAO-F, CCAR-P).

Every folder here is referenced from a specific blog article and, where the article has a gated downloadable artifact (cheat sheet, template), from that PDF too. The code is meant to be read and run, not just skimmed — clone it, follow the README in each folder, and compare it against what the exam guide actually asks for.

## Why this exists

The exam guides describe architecture decisions in the abstract. This repo makes them concrete: real MCP servers, real agent loops, real config files — the kind of thing you'd actually be graded on being able to build or reason about.

Free practice exams for all four certifications: [aicert.study](https://aicert.study).
