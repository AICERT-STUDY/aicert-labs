# CCAO-F — Claude Certified Associate, Foundations

Planned for wave 3. This certification is aimed at non-technical Claude users, so its artifacts won't be code — expect prompt template libraries, an output-evaluation checklist, a governance red-flags cheat sheet, and use-case playbooks, one per official domain (Output Evaluation and Validation, Workflow Integration and Solution Design, Governance/Risk/Responsible Use, Prompting and Task Execution, Product and Model Selection, Configuration and Knowledge Management, Troubleshooting and Optimization).

Full guide and study material in the meantime: [aicert.study/certifications/ccao-f](https://aicert.study/en/certifications/ccao-f).
